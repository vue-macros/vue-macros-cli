# vue-macros-sg

vue-macros-sg is a cli for rewriting at vue macros powered by ast-grep.

## Support vue macros

- [x] jsx-directive
- [x] define-render
- [x] export-render
- [x] define-slots
- [x] short-v-model
- [x] setup-sfc 

## Usage

```shell
npx vue-macros-sg ./src
```
