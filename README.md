# @vue-macros/cli

@vue-macros/cli is a CLI for rewriting at Vue Macros powered by [ast-grep](https://github.com/ast-grep/ast-grep).

## Support vue macros

- [x] jsx-directive
- [x] define-render
- [x] export-render
- [x] define-slots
- [x] short-vmodel
- [x] setup-sfc 


## Usage

```shell
# install
pnpm add -g @vue-macros/cli
```

### SG
```shell
vue-macros sg
```

### Init
```shell
pnpm create vite my-vue-macros --template vue-ts

cd my-vue-macros

vue-macros init
```
