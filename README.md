# @vue-macros/cli

@vue-macros/cli is a cli for rewriting at vue macros powered by [ast-grep](https://github.com/ast-grep/ast-grep).

## Support vue macros

- [x] jsx-directive
- [x] define-render
- [x] export-render
- [x] define-slots
- [x] short-v-model
- [x] setup-sfc 


## Usage

```shell
# install
pnpm add -g @vue-macros/cli
```

### sg
```shell
vue-macros sg
```
